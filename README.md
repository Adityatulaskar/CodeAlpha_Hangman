# 🎮 Hangman Game in Python

A fun and interactive command-line Hangman game built using Python. This project was developed as part of the CodeAlpha Internship program and demonstrates core Python programming concepts such as loops, conditionals, functions, lists, dictionaries, and user input handling.

---

## 📌 Features

- 🎯 Multiple difficulty levels:
  - Easy
  - Medium
  - Hard

- 💡 Smart hint system
- 🏷️ Word categories
- ✨ First-letter reveal for Easy and Medium modes
- 🎨 ASCII Hangman visualization
- 🔄 Play Again option
- ✅ Input validation
- 📊 Wrong guess tracking

---

## 🛠️ Technologies Used

- Python 3
- Random Module
- Command Line Interface (CLI)

🎮 Gameplay

Select a difficulty level.
Guess letters one by one.
Avoid incorrect guesses to keep the hangman from being completed.
Use hints and categories strategically.
Guess the word before running out of attempts.
